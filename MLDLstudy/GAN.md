# GAN (Generative Adversarial Networks)
- 
## 개념과 핵심용어
- Discriminator : 분별자 
    - 실제값과 Generator가 생성한 값을 비교함..
    - Discriminator가 분별하지 못할때 까지 학습
- Generator : 생성자
    - DL Encoder/Decoder 에서 Decoder로 -> 가상 값(이미지)를 생성

## 모델구성
- Generator가 만든 x페이크 
- Discriminator에 z로 생성

## 손실함수
- 최소화/최대화를 같이 쓰는게 이해가 안된다..
